🚀 Proyecto Destacado

🤖 Bot de Telegram conectado a instancias EC2 de AWS

Proyecto desarrollado como trabajo de fin de grado. Implementé un bot de Telegram creado con Next.js que se conectaba de forma remota a una instancia EC2 de Amazon Web Services. El bot podía ejecutar comandos a distancia sobre la máquina virtual (por ejemplo, reiniciar servicios, consultar estado o ejecutar scripts), permitiendo administrar el servidor directamente desde Telegram.

Este proyecto me permitió integrar desarrollo web, seguridad y automatización en la nube, utilizando la API de Telegram y la infraestructura de AWS.
